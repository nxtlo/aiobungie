Bungie profiles requires components. But its kinda boring to pass multiple components to a parameter. So. The .Profile crate will include all Bungie components. to be accessiable as a crate.

How?. For an example: to access the Characters component you'll need to pass ?component=200. But here you can just return the character itself from the profile using await .ProfileComponent.titan() and the other character methods which returns a Character crate. crates are basically classes/objects.

client = aiobungie.Client(...)
profile = await client.fetch_profile("Fate")
